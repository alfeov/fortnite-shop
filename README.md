# Fortnite Shop

- Project for shopping Fortnite game items with [Fortnite-API](https://fortnite-api.com/)
- In project used:
  - React
  - React Context
  - Vite
  - SCSS
- ! Important: this project has fetch delay for showing loaders, Infinite Scroll Implementation and cashing data. You can turn it off in src/api/getMovies
